Attaching a Socketable

Takes 1 hour and requires proficiency in one of: Carpenter's, Cobbler's, Glassblower's, Leatherworker's, Smith's, Tinker's, Weaver's, or Woodcarver's tools.

All items of Common rarity or higher have 1 socket. Some socketables require the host item to be attuneable.

Removing a Socketable

Requires the same tool proficiency.

Roll a DC 10 Dexterity or Intelligence check using the tool. On a failure, the socketable item is destroyed. The host item is unaffected.

Resolve Removal

DC 10 Dex or Int check with appropriate tool proficiency. Failure destroys the socketable.
